Portafolio Tecnológico

Hola, bienvenido a mi portafolio... A continuación podrás encontrar proyectos realizados de forma personal para incrementar mis conocimientos sobre diseño y desarrollo web, inicialmente se empezaron publicando en la pagina de netlify.com, con el fin de contar con un servidor que me permita cargar mis proyectos de forma gratuita, eventualmente con el uso de GIT y Git Hub se estarán cargando los proyectos y cambios de los mismos por medio de un git push a Git Hub para contar con un servidor remoto, dejo los tres primeros proyectos elaborados para ver mas puedes visitar mi perfil en GitHub.

EMDev

Este fue mi primer proyecto real en HTML y CSS, aqui plasme en una web lo que quería expresar sobre .mi, seguramente vendran algunos cambios conforme aprenda un poco mas, creo que para ser mi primer proyecto me quedo muy bien, podras ver diferentes pestañas acerca de lo que soy yo, mis intereses y portafolio, podras descargar mi libro, escribirme y ver mis redes sociales.

Make UP Store

Pagina en donde me permiti, poder enfocar un poco mas los conocimientos adquiridos en CSS distribuyendo imagenes, controlando mas el CSS Grid y dandole un estilo caracteristicon con una paleta de colores algo mas vistosos para la idea de una tienda algo mas atractiva de maquillaje que las convencionales, fue el boceto para iniciar un proyecto familiar de venta de productos para belleza.

BlogDeCafé

Aqui, podemos ver un poco mas de orden sobre los conceptos previos adquiridos sobre HTML y CSS, con este proyecto inicie mis primeros pasos en JS, se pueden ver mejoras en el control responsivo de la misma pagina WEB permitiendo ver una visualizacion en un dispositivo con cualquier tamaño de pantalla, tambien aqui se utilizo constantemente GIT, anotando los respectivos cambios por medio de Commits, no se usaron ramas puesto que no era un trabajo en equipo era yo mismo contra el mundo, no se ha cargado en GitHub por uun pequeño error que sera solucionado prontamente.
